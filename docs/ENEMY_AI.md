# おじさんX — ENEMY AI（敵AI・ボス攻撃の挙動）

作成日: 2026-06-26

> 各敵・ボスが「いつ・どう攻撃するか」を確定する。判定・タイミング・上段/下段まで。
> 状態の枠は `STATE_MACHINES.md`、数値は `TUNING.md` を参照。

---

## 0. 共通の考え方

- 敵は `decide()` で次の行動を決める。基本は「**間合い** と **クールタイム** と **乱数**」で判断。
- **上段/下段（high/low）属性**：攻撃・弾に `level: 'high' | 'low'` を持たせる。
  - high＝プレイヤーの上半身に当たる。**しゃがみで回避**できる
  - low＝足元に当たる。**ジャンプで回避**できる
  - プレイヤーの攻撃も同様：立ち攻撃=high、しゃがみ攻撃=low、で敵の高さ属性に対応
- 「**攻撃後の硬直（recover）**」が崩しポイント。硬直を長くすると簡単・短くすると難しい。
- **本文中の数値（間合い・クールタイム・前隙・硬直）は `TUNING.md` の `ENEMY_BEHAVIOR` / `BOSS_BEHAVIOR` が正本**。
  ここの数字は読みやすさのための再掲なので、調整する時はTUNINGを更新する（二重管理しない）。

## 1. ハグ魔（つかみ敵）

**目的**：プレイヤーに密着して体力を吸う。直接攻撃でなく「拘束」で脅かす。

```
decide():
  距離 > grabRange(60px) → approach（まっすぐ近づく・速度110）
  距離 <= grabRange かつ recover中でない → attack(grab) へ
```

- `attack(grab)`：プレイヤーが `hurt/dead/ハゲ化` でなければ `grabbed` 状態に拘束
  - 拘束成立中：プレイヤー体力を `drainPerSec`(2/s) で吸う。複数体で加速
  - プレイヤーが連打 `mashToEscape`(6) 到達 → 振りほどき。ハグ魔は `escapeKnockback`(160) で吹っ飛び `recover`
  - 振りほどかれたハグ魔は1秒 `recover`（再つかみ不可）
- **倒し方**：つかまれる前に攻撃で潰す。立ち/しゃがみどちらでも当たる（高さ属性なし＝接近型）
- **嫌らしさ調整**：複数同時接近で焦らせる。ただしF1では1体ずつ出して学習させる

## 2. 名刺投げ（遠距離敵）

**目的**：距離を保ち、上段/下段の名刺で揺さぶる。

```
decide():
  距離 < tooClose(140px) → 後退（プレイヤーから離れる）
  throwRange(140〜420px) かつ クールタイム(throwCd 1500ms)経過 → attack(throw)
  それ以外 → idle/微移動
```

- `attack(throw)`：50%でhigh（頭の高さ）、50%でlow（膝の高さ）を投げ分け
  - 弾速 `projectileSpeed`(260)。highは直線、lowは少し落ちる軌道でも可
  - プレイヤーは high→しゃがみ、low→ジャンプ or 立ち回避
- 投げモーションに前隙（windup 250ms）。ここを読んで接近・潰せる
- **倒し方**：弾を捌いて接近、近接で潰す。近づくと後退するので画面端に追い込む

## 3. ちびリーマン（飛びかかり敵・小型）

**目的**：素早く距離を詰めて飛びかかる。低姿勢で立ち攻撃が当てにくい。

```
decide():
  距離 > jumpRange(180px) → approach（高速150で走る）
  距離 <= jumpRange かつ クールタイム経過 → attack(leap)
```

- `attack(leap)`：プレイヤーへ放物線ジャンプ。空中＝high扱い、着地際＝low
  - ジャンプ中の敵には **プレイヤーのジャンプ攻撃** が刺さる（読み合い）
  - 着地後 `recover`(500ms)。ここを潰す
- 小型でしゃがみ攻撃（low）が当てやすい／立ち攻撃(high)は空振りしやすい
- **倒し方**：飛んできたらジャンプ攻撃で迎撃 or 着地硬直に下段

## 5. ボス攻撃パターン詳細

### F1 傘おじさん（HP8）
```
idle(クールタイム 800ms) → move(間合い調整) → 攻撃抽選:
  60% attack1=突き（umbrella thrust）: リーチ長(120px)・high・前隙300ms・硬直500ms
  40% attack2=払い（sweep）: 中リーチ(90px)・low・横振り・硬直400ms
```
- 崩し：突きの**前隙**を見て一歩引く→**硬直500ms**に踏み込んで2〜3発
- 学習設計：F1ボスなので硬直長め・パターン2種でシンプル

### F2 ブーメラン部長（HP10）
```
idle → attack1=ファイル投げ: 往復するブーメラン弾
  往路は前方へ、一定距離で戻ってくる（プレイヤーの背後を通る軌道）
  high固定。投げ後 recover 600ms。本体は接近に弱い
```
- 崩し：往路を避け、戻り弾に注意しつつ接近。弾を捌くより本体を殴る方が早い

### F3 筋肉専務（HP14・巨体 displayHeight 280）
```
idle → 抽選:
  50% attack1=つかみ（grab）: 接近して掴む→連打で脱出（ハグ魔の強化版・ダメージ大）
  50% attack2=突進（charge）: 一気に走って体当たり・地響きでプレイヤーよろけ
```
- 突進は予備動作（足踏み400ms）あり→**空かして側面へ**。突進後の硬直(700ms)に攻撃
- つかみは掴まれたら連打で脱出（脱出失敗で大ダメージ）
- 巨体なので**全ポーズで足元をフレーム下端に揃える**（床めり込み厳重注意・PHASER3 #1）

### F4 占い師おじさん（HP12）＋ 大仏豚（中ボス HP8）
```
占い師: idle → attack1=詠唱: 水晶玉から火の玉を放つ
  火の玉は high/low をランダム。発射後一定距離で「変化」:
    → 蛇（low・地を這う）/ 蛾（上下にスウェーブ）/ おやつ（直進・遅い）
  詠唱中(windup 500ms)は本体無防備＝チャンス
```
- 大仏豚（中ボス）は道中の関門。既存 `boss/` 資産流用。接近＋落下物程度のシンプルパターン
- 崩し：詠唱の隙に近接。変化弾は高さを見て上段/下段でガード移動

### F5 Mr.X（会長・ラスボス）第1形態HP12／第2形態+10
```
第1形態: idle → 抽選:
  attack1=ステッキ薙ぎ（high・中リーチ・硬直中）
  attack2=回し蹴り（low・接近時）
HP<=50% → phase2移行演出（無敵2秒・画面演出）→ 第2形態:
  攻撃速度UP（クールタイム短縮）＋ attack3=連続コンボ（薙ぎ→蹴り）追加
```
- 設計意図：これまでの全要素（接近・遠距離・読み合い）の総決算
- **ハゲ化アイテムを第2形態にぶつける**のが想定攻略（プレイヤーに選択を促す）

## 6. 難易度カーブ（フロア横断の設計）

| F | 同時出現数 | 主な要求スキル | ボス硬直 |
|---|---|---|---|
| 1 | 1〜2 | 基本操作・つかみ脱出 | 長い（易） |
| 2 | 2〜3 | 上下回避・遠近同時処理 | 中 |
| 3 | 2〜3 | 突進対応・密度 | 中 |
| 4 | 2〜3 | 遠距離弾捌き・中ボス | やや短 |
| 5 | 3 | 総合 | 短い（難） |

- 「途中で進めない」時は **同時出現数 → 敵HP → ボス硬直** の順に緩める（TUNING調整指針と対応）。

## 7. 実装メモ

- 各敵の `decide()` は「距離・クールタイム・乱数」の単純ルールで十分。凝らない
- 攻撃には必ず `level('high'|'low')` を持たせ、プレイヤーの回避（しゃがみ/ジャンプ）と噛み合わせる
- ボスの攻撃抽選は重み付きランダム（上の%）。`recover` 時間がそのまま難易度
- 弾は共通の `Projectile` クラスに `level` と `transform`（変化）オプションを持たせて使い回す
