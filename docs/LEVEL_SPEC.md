# おじさんX — LEVEL SPEC（全5フロアの中身）

作成日: 2026-06-26

> 各フロアの敵配置・ハザード・ボスを定義。これが `src/data/floors.js` の元になる。
> 数値（HP・ダメージ・時間）は `TUNING.md` を正とし、ここでは構成を書く。

---

## 共通ルール（全フロア）

- 横スクロールで右へ進む。ワールド右端＝ボス部屋。
- 敵は「x座標トリガー」で出現（プレイヤーがx地点を超えるとそのウェーブをspawn）。
- 同時出現数に上限（処理負荷と難度のため。`TUNING.md`）。
- フロア踏破の流れ：**雑魚ウェーブ複数 → ボス部屋でカメラロック → ボス撃破 → 札束/階段で次フロアへ**。
- 制限時間あり。0で1ミス。

### ミス時のリスポーン仕様（共通）
- **HP0 または 時間切れ＝1ミス**。残機を1減らす。
- **リスポーン位置**：直近に通過した「チェックポイントx」から再開（フロアを丸ごと最初からにはしない）。
  - チェックポイントは各ウェーブのトリガーx地点に設定（ウェーブを1つ越えるごとに更新）。
  - ボス部屋に入っていた場合はボス部屋入口から再開し、ボスHPは全回復。
- リスポーン時：プレイヤーHP全回復・短い無敵（リスポーン硬直）・制限時間はそのフロアの規定値にリセット。
- **残機0でミス → ゲームオーバー**（ResultSceneへ）。
- ※MVP(F1)では「チェックポイント＝フロア頭」の最簡版でも可。仕様を固定しておき、実装は段階的に。

### ハゲ化アイテム（itemPowerup）の入手ルール
- **配置方式**：各フロアに**固定で1個**、ボス部屋の少し手前に出現（ボス戦に持ち込ませる設計意図）。
  - 加えて、特定の雑魚を倒すと**低確率でドロップ**（`TUNING` `ITEM.powerupDropRate`）。
  - F1はチュートリアルとして「ボス前に必ず1個」を見える位置に置く。
- 取得で即発動（`POWERUP.durationMs` 8秒）。発動中の重ね取りは時間延長。
- 1フロアに出しすぎない（ありがたみが消える）。基本は固定1個＋低確率ドロップ。

### エッジケースの扱い（実装時に迷わないよう明文化）
| 状況 | 挙動 |
|---|---|
| **つかまれ中に時間切れ** | 通常どおり1ミス。拘束解除してリスポーン処理へ |
| **ボス戦中に時間切れ** | 1ミス。リスポーンはボス部屋入口・**ボスHPは全回復**（リスポーン仕様と同じ） |
| **ハゲ化中に時間切れ** | 時間切れはハゲ化より優先＝1ミス（無敵は被弾には効くが時間には効かない） |
| **穴に落下（F3等）** | MVPでは穴を作らない。作る場合は「小ダメージ＋直前の足場へ押し戻し」。即ミスにはしない |
| **ボス撃破と同時にプレイヤーHP0** | プレイヤー優先で判定せず、**ボス撃破を優先**（理不尽な相打ち死を防ぐ） |
| **画面外に吹き飛んだ敵** | ワールド境界で消滅（撃破扱い）。プレイヤーは境界でクランプ（落下しない） |

## FLOOR 1 — 居酒屋街（チュートリアル兼）

- **ワールド幅**：3600px / **制限時間**：90秒 / **BGM**：bgmStage
- **テーマ**：赤提灯・居酒屋の並ぶ路地。下層のにぎやかさ。
- **登場雑魚**：ハグ魔（つかみ）＋名刺投げ（遠距離）
- **狙い**：基本操作・つかみ振りほどき・上段下段の打ち分けを自然に学ばせる
- **ウェーブ例**：
  | トリガーx | 出現 |
  |---|---|
  | 400 | ハグ魔 ×1（つかみを体験させる） |
  | 900 | 名刺投げ ×1（遠距離をジャンプ/しゃがみでかわす） |
  | 1500 | ハグ魔 ×2 |
  | 2200 | 名刺投げ ×1 ＋ ハグ魔 ×1 |
  | 2900 | ハグ魔 ×2（ボス前の関門） |
- **ボス：傘おじさん**（リーチの長い突き・払い。間合い管理を学ぶ）

## FLOOR 5 — 社長室（屋上・最終）

- **ワールド幅**：3200px（短めの最終決戦ロード） / **制限時間**：110秒
- **テーマ**：夜景の見える最上階・社長室。
- **登場雑魚**：これまでの全種が少数ずつ登場（総力戦）
- **狙い**：総決算。全ての敵タイプへの対応を試す
- **ウェーブ例**：
  | トリガーx | 出現 |
  |---|---|
  | 500 | ハグ魔 ×2 ＋ 名刺投げ ×1 |
  | 1300 | ちびリーマン ×2 ＋ バナナ筋肉女子 ×1 |
  | 2100 | フルーツ眼鏡女子 ×1 ＋ 名刺投げ ×2 |
- **ラスボス：Mr.X（会長）**
  - **第1形態**：ステッキ薙ぎ（上段/high）＋回し蹴り（下段/low）の2種（TUNING `BOSS_BEHAVIOR.mrx`・ENEMY_AI・STATE_MACHINESと統一）
  - **第2形態**：HP50%以下で「怒りの会長」へ変身。クールタイム短縮＝攻撃速度UP＋連続コンボ（薙ぎ→蹴り attack3）を追加
  - ※「全ボスの技を複合」という当初メモは取りやめ（実装が複雑化し他ファイルと矛盾するため）。総決算感は道中ウェーブ（全敵タイプ登場）と第2形態のコンボで表現する
  - 撃破で退職金（itemCash）奪還 → 全クリア演出

## ボス攻略パターン早見表

| ボス | 主攻撃 | 弱点・崩し方 |
