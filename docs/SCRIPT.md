## 1. タイトル画面

- ロゴ：**おじさんX**
- サブ：〜退職金奪還〜
- 点滅：`PUSH SPACE`
- ハイスコア：`ハイスコア  012300`
- 操作（小さく下部）：
  `←→ 移動　↑ ジャンプ　↓ しゃがみ　Z パンチ　X キック`

## 2. オープニング（挑戦状）

ゲーム開始時に短く表示（スキップ可）。

```
ある朝、オジサンホックスの机に一通の手紙が。

「貴殿の退職金、確かに預かった。
　取り返したくば、X商事ビルを登ってこい。

ふざけるな。40年、働いてきたんだ。
オジサンは、ブリーフケースを握りしめた。
```

## 7. エンディング（全クリア）

Mr.X撃破後：

```
オジサンは、ついに退職金を取り戻した。

長い長い、ビルだった。
だが、登りきった先に待っていたのは──

「お父さん、おかえり」

家族の待つ、いつもの食卓だった。

	おじさんX  ─ 完 ─

	スコア  042300
```
> ※エンディングの落とし所（家族／第二の人生／実は夢オチ 等）は好みで差し替え可。
> 「退職金の先に家族がいた」案を仮で置いている。

## 8. トーンの注意

- コミカルで哀愁あるサラリーマン哀歌。シリアスにしすぎない。
- 漢字は読みやすく。難読語は避ける（ファミコン世代も読める平易さ）。
- 文言は短く。画面に長文を出さない（テンポ優先）。
- 文字列は実装時 `src/data/text.js` 等にまとめ、コード直書きしない（後で推敲・多言語化しやすく）。
