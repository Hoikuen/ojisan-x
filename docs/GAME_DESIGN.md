## 1. コンセプト（1〜3行）

リストラ寸前のサラリーマン「オジサンホックス」が、黒幕 **Mr.X（謎の会長）** にさらわれた
**大事なもの（退職金＝札束）** を取り返すため、魔の摩天楼「**X商事ビル**」を1階から登る。
各フロアの雑魚を**ブリーフケース**でなぎ倒し、フロアボスを倒して上へ。最上階のMr.Xを倒せばクリア。

**元ネタ**：『スパルタンX』(Kung-Fu Master, Irem 1984)。5フロアの塔を登り、各階のボスを倒す構造を踏襲。
キャラ・世界観・演出は完全オリジナル（おじさん/サラリーマン世界）に置き換える。

---

## 2. 「何が楽しいか」（コアの面白さ）

1. **テンポよく敵をなぎ倒す爽快感** — 連打で連続攻撃、ジャンプキックで飛んでくる敵を落とす
2. **つかみ系敵のヒリつき** — ハグ魔につかまれると体力が吸われる→振りほどく緊張感
3. **ボスごとの攻略パターン** — 5体のボスがそれぞれ違う攻撃。覚えて崩す
4. **ハゲ化（怒りモード）の一発逆転** — ピンチでパワーアップアイテムを取ると一定時間無敵級
5. **登っていく達成感** — フロアが上がるごとに背景・敵・難度が変わる

---

## 3. 物語・世界観

- **舞台**：雑居ビル「X商事ビル」（5フロア＋屋上）。下層は居酒屋街、上に行くほど怪しい会社のオフィス、最上階は社長室。
- **発端**：オジサンホックスの机に挑戦状。「お前の退職金は預かった。取り返したくば登ってこい ─ Mr.X」
- **目的**：5フロアを登り、各フロアのボス（5人の刺客）を倒し、最上階のMr.Xから退職金を奪還する。
- **トーン**：コミカルで哀愁あるサラリーマン哀歌。シリアスになりすぎず、おじさんギャグを散りばめる。

## 4. 操作（コントロール）

| 入力 | 動作 |
|---|---|
| ← → | 移動（左右） |
| ↑ | ジャンプ |
| ↓ | しゃがみ（下段攻撃をかわす／下段攻撃の起点） |
| Z または Space | パンチ（ブリーフケースで殴る・**上段/high**） |
| X | キック（立ちキック・**上段/high**） |
| ↓ + Z/X | 下段攻撃（しゃがみ攻撃・**下段/low**） |
| ジャンプ中 + Z/X | ジャンプ攻撃（飛びかかる敵に有効） |
| Z/X 連打 | 連続攻撃（その場で高速連打） |

- パッド対応も将来的に視野（MVPはキーボード）。
- スパルタンX同様、**上段(high)/下段(low)**の打ち分けが攻略の肝。
  - **立ち攻撃（パンチ・立ちキック）＝上段/high** … 上半身に来る敵・high弾に対応
  - **しゃがみ攻撃（↓+Z/X）＝下段/low** … 足元の小型敵・low弾に対応
  - ※「キックが下段」ではない。下段はしゃがみ攻撃で出す（ENEMY_AI/TUNINGと統一）。

## 5. 登場キャラクター

### 主人公：オジサンホックス
- リストラ候補のサラリーマン。スーツ＋ブリーフケース。
- 攻撃はブリーフケースで殴る／蹴る。
- **パワーアップ＝ハゲ化（怒りモード）**：アイテムで一定時間、攻撃力UP＋無敵級＋見た目がハゲる。
- 既存資産：`spartan-hex/player_ojisan/`（idle walk attack jump fall hurt extra）＝**流用可**。ハゲ版は`player_bald/`（jump/fall/hurt不足）。

## 6. ゲーム進行・勝敗条件

- **全5ステージ**。各ステージ：横スクロールで右へ進む → ボス部屋 → ボス撃破 → 次フロアへ。
- **プレイヤーHP**：体力バー制（連続ダメージで減少／つかまれると継続減少）。残機制も併用（初期3機）。
- **時間制限**：各フロアに制限時間（スパルタンX同様）。0で1ミス。
- **クリア条件**：F5でMr.Xを撃破。
- **ゲームオーバー**：残機0でゲームオーバー。
  - **コンティニュー**：ResultSceneで一定時間「CONTINUE? (Spaceで継続)」を表示。継続で**そのフロア頭から残機リセット再開**（スコア引き継ぎ・回数制限なし＝家庭用寄り）。押さなければスコア表示→タイトル。※MVPは「継続なし→タイトル」最簡版でも可。
- **残機追加（エクステンド/1UP）**：スコアが `SCORE.extendThreshold`（例:30000）到達ごとに1UP（ストック上限あり）。
- **ループ（任意）**：全クリ後に難度を上げて周回（MVPでは未実装でよい）。

詳細な敵配置・時間・HPは `LEVEL_SPEC.md` と `TUNING.md`。

## 9. MVP（最初に完成させる範囲）＝vertical slice

**「F1を最初から最後まで遊べる」**を最初のゴールにする。
- F1の背景・雑魚（ハグ魔＋名刺投げ）・ボス（傘おじさん）
- 操作・攻撃・しゃがみ・ジャンプ攻撃・体力・時間・ミス・クリア遷移
- タイトル → F1 → クリア → （仮の）次フロア表示 → ゲームオーバー → タイトル

ここが完成し、手触りがOKになってからF2〜F5へ横展開する（敵・ボス・背景を差し替える形）。

---

## 10. やらないこと（MVPスコープ外）

- オンライン要素・セーブデータ（ハイスコアのみ）
- 複雑なコンボ表・必殺技ゲージ
- 周回ループの難度変化（最初は1周クリアで完成扱い）
- 演出過多（凝ったカットシーンは後回し）
