# おじさんX — UI FLOW（画面遷移・UI/HUD設計）

作成日: 2026-06-26

> 画面の流れと、ゲーム中のHUD配置を確定する。キャンバスは800×600。

---

## 1. 画面遷移フロー

```
            ┌──────────────────────────────────────┐
            ▼                                      │
        [TitleScene]                               │
            │ スタート押下                          │
            ▼                                      │
        [GameScene] ── フロア1〜5を順に ──┐         │
            │                            │         │
   フロアクリア│              残機0 / 全クリア│         │
            ▼                            ▼         │
    （次フロアへ自動continue）      [ResultScene]    │
                                    │ ・全クリア     │
                                    │ ・ゲームオーバー │
                                    │ ・ハイスコア表示 │
                                    └── タイトルへ ──┘
```

- ポーズ（Pキー）：GameSceneにオーバーレイ表示。シーン遷移はしない。

---

## 2. 各画面の中身

### TitleScene
- 背景：X商事ビルの外観（bgTitle）
- タイトルロゴ「おじさんX」（大きく中央上）
- 「PRESS SPACE TO START」点滅
- 操作説明（小さく下部）：←→移動 / ↑ジャンプ / ↓しゃがみ / Zパンチ / Xキック
- ハイスコア表示（localStorageから）

### GameScene（本編）— HUD配置

```
┌─────────────────────────────────────────────────────────┐
│ ❤❤❤❤❤❤❤❤□□□□□□□□   SCORE 012300   ⏱ 087   🏢 1F │ ← 上部HUD(高さ約40px)
│ [1Pライフバー]          [スコア]      [時間]   [フロア]    │
│                                                         │
│                                                         │
│                  （ゲームプレイ領域）                      │
│                                                         │
│              🧍‍♂️オジサンホックス      🧟敵                 │
│─────────────────────────────────────────────────────────│ ← 床(FLOOR_Y=575)
│ 残機: 🧍×2                          [ボス戦時↓]          │
└─────────────────────────────────────────────────────────┘
            ボス戦：画面上部にボスのライフバー（赤）＋ボス名
```

| 要素 | 位置 | 内容 |
|---|---|---|
| プレイヤーライフバー | 左上 | 16目盛のハート/ゲージ。減ると赤→点滅 |
| スコア | 中央上 | 数字（ゼロ詰め6桁） |
| 残り時間 | 右寄り上 | 秒カウントダウン。残り10秒で赤点滅 |
| フロア表示 | 右上 | 「1F」〜「5F」「RF（屋上）」 |
| 残機 | 左下 | アイコン×残数 |
| ボスライフバー | 上部中央（ボス戦のみ） | 赤バー＋ボス名。撃破でアニメ消滅 |
| ハゲ化ゲージ | プレイヤー頭上 or 左上 | 発動中のみ残り時間バー表示 |

- HUDは `setScrollFactor(0)`（カメラ追従しても固定）。
- フロア開始時に中央へ「FLOOR 1 ─ 居酒屋街」を2秒オーバーレイ表示。

### フロアクリア（GameScene内オーバーレイ→ResultScene簡易版）
- 「FLOOR CLEAR!」表示
- 残り時間ボーナス加算アニメ（時間×50pt）
- 数秒後に自動で次フロアへ（continue）

### ResultScene
- **全クリア時**：「退職金を取り戻した！」演出＋総スコア＋ハイスコア更新表示
- **ゲームオーバー時**：「GAME OVER」＋到達フロア＋スコア＋ハイスコア
- 「PRESS SPACE → TITLE」

### ポーズ（オーバーレイ）
- 半透明黒幕＋「PAUSED」＋「P:再開 / R:リスタート / T:タイトル」

---

## 3. UI演出（細部）

- スコア加算は「カウントアップ」させる（パッと変えず転がす）
- 被弾でライフバーが減るときは一瞬白フラッシュ
- 残り時間10秒以下：数字赤＋点滅＋SE（チクタク）
- ハゲ化発動：画面ふちが一瞬光る＋ゲージ出現
- フロア表示・FLOOR CLEAR・GAME OVERは大きめの太字でドンと出す（ファミコン感）

---

## 4. フォント・色（トンマナ）

- フォント：太めのレトロ系（ピクセルフォント or 太ゴシック）。日本語が出るものを選ぶ
- 色：
  - ライフ＝赤系、時間＝黄→赤、スコア＝白、フロア＝水色
  - HUD背景は半透明の黒帯（視認性確保）
- 全体トーン：明るくコミカル。シリアスにしすぎない

---

## 5. レスポンシブ／表示

- `Phaser.Scale.FIT` で画面に収める（アスペクト維持・余白は黒）
- HUDは800×600基準の絶対座標で配置（FITで一緒に拡縮）

---

## 6. 実装メモ

- HUDは専用の `UiManager`（または GameScene内のメソッド群）で一元更新。`updateUi()` を毎フレーム or 値変化時に呼ぶ
- ボスライフバーは `bossLock` モード移行時に生成、`dead` で破棄
- 画面中央テキスト（FLOOR表示等）は使い回す共通ヘルパー `showCenterText(text, durationMs)` を作る
- localStorage キー：`ojisanx_highscore`
